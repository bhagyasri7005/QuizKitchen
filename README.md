# QuizKitchen

QuizKitchen is a full-stack web application for generating quizzes and managing quiz results.

## Features
- Generate quiz questions
- View quiz results
- Backend API for quiz handling
- Client-server architecture

## Tech Stack
Frontend: React / HTML / CSS / JS  
Backend: Node.js / Express  

## Project Structure
client/  -> Frontend  
server/  -> Backend

## How to Run

1. Clone the repository
git clone https://github.com/bhagyasri7005/QuizKitchen.git

2. Install dependencies
npm install

3. Run server
npm start
